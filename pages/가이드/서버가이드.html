서버가이드

서버 이용에 필요한 핵심 정보만 정리했습니다. (필요하면 이 페이지 내용을 더 채워갈 수 있어요.)

1) 기본 정보

  • 서버 규정: 서버규정 페이지
  • 점검 안내: 서버점검 페이지

2) 시작 전 체크

  • 클라이언트/패치 적용 여부 확인
  • 접속/계정 관련 안내 확인
  • 문제 발생 시 공지사항 먼저 확인

3) 자주 묻는 내용

  • 접속이 안 될 때: 점검/공지 확인 → 재시도
  • 게임 내 이상 현상: 공지 확인 후 재접속
  • 규정 위반/제재 관련: 서버규정 참고

※ 이 페이지는 안전하게 기본 틀만 추가한 상태입니다. 필요하면 항목을 더 추가해 확장할 수 있습니다.
