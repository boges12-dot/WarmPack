2단계 마법

디크리즈 웨이트 소모: MP : 0

설명: 1800초 동안 소지 무게 한도가 +180 증가한다.

디텍션 소모: MP : 0

설명: 화면에 숨어있는 PC 또는 NPC를 찾아낸다.

인첸트 웨폰 소모: MP : 0

설명: 1800초 동안 선택한 근거리 무기에 근거리 대미지 +2 효과 적용.

칠터치 소모: MP : 0

설명: 대상의 HP를 소량 흡수하여 자신의 HP로 전환한다. 사용 시 성향치 감소.

커스 포이즌 소모: MP : 0

설명: 30초 동안 대상에게 독 상태를 부여, 지속적으로 HP가 감소한다.

큐어 포이즌 소모: MP : 0

설명: 자신 또는 대상의 중독 상태를 해제한다.

파이어 애로우 소모: MP : 0

설명: 불꽃의 마법 화살을 날려 대상에게 대미지를 준다.

3단계 마법

라이트닝 소모: MP : 0

설명: 술자와 대상 사이 직선상의 모든 적에게 피해를 준다. 갑옷에 사용 시 1800초 동안 AC-3 효과 적용.

블레스트 아머 소모: MP : 0

설명: 연속 클릭 시 착용 중인 갑옷에 적용된다.

익스트라 힐 소모: MP : 0

설명: 대상의 HP를 약간 회복시킨다. 언데드 몬스터에게는 피해를 준다.

프로즌 클라우드 소모: MP : 0

설명: 공격 대상 주위에 냉기를 퍼뜨려 주변 대상에게 피해를 준다.

커스 블라인드 소모: MP : 0

설명: 대상을 암흑 상태로 만든다.

턴 언데드 소모: MP : 0

설명: 일정 확률로 언데드 몬스터를 죽이거나 버서크 상태로 만든다.

4단계 마법

메디테이션 소모: MP : 0

설명: 평소보다 5+@만큼 MP를 더 회복한다. 이동, 전투 시 효과 사라짐.

뱀파이어릭 터치 소모: MP : 0

설명: 대상의 HP를 상당량 빼앗아 자신의 것으로 만든다.

슬로우 소모: MP : 0

설명: 대상의 이동 및 공격 속도를 느리게 한다.

어스 재일 소모: MP : 0

설명: 주변 2칸 내 적을 가시덩쿨로 공격한다.

웨폰 브레이크 소모: MP : 0

설명: 대상 무기를 일정 확률로 손상시킨다.

인첸트 덱스터리 소모: MP : 0

설명: 1200초 동안 대상에게 DEX+5 효과 적용.

카운터 매직 소모: MP : 0

설명: 16초 동안 한번만 마법 무효화, 일부 마법 제외.

파이어 볼 소모: MP : 0

설명: 불타는 화염구가 주변 3칸 내 적에게 피해.

5단계 마법

그레이터 힐 소모: MP : 0

설명: 대상의 HP를 꽤 회복시킨다. 언데드에게는 피해.

설명: 대상의 시야를 완전히 없앤다. 캔슬레이션/리무브 커스로 해제 가능.

리무브 커스 소모: MP : 0

설명: 대상의 저주를 일정 확률로 해제. 일부 저주 제외.

마나 드레인 소모: MP : 0

설명: 대상의 MP를 흡수하여 자신의 MP로 전환.

커스 패럴라이즈 소모: MP : 0

설명: 대상을 일정 확률로 마비시킨다.

테이밍 몬스터 소모: MP : 0

설명: 몬스터에게 최면을 걸어 명령을 따르게 한다. CHA와 연관 있음.
