클라이언트 다운로드

아래 버튼을 눌러 클라이언트를 다운로드하세요. 링크는 운영자가 실제 주소로 교체하면 됩니다.

클라이언트 다운로드 패치/업데이트 파일
  • 권장: Windows 10/11
  • 압축: ZIP (해제 후 실행)
  • 용량: 운영자 기입

설치 · 실행 방법

  1. 다운로드한 ZIP 파일을 압축 해제합니다.
  2. 폴더 안의 Launcher / Game 실행 파일을 실행합니다.
  3. 최초 실행 시 방화벽 안내가 뜨면 허용을 선택합니다.
  4. 접속 정보/계정 생성은 서버규정 또는 공지에서 안내합니다.
공지사항 보기 서버 가이드

자주 발생하는 오류

실행이 안 되거나 바로 꺼져요
  • 압축을 완전히 해제했는지 확인하세요(바탕화면 임시폴더에서 실행 X).
  • 백신/보안 프로그램이 차단하는 경우, 게임 폴더를 예외로 등록해보세요.
  • 관리자 권한으로 한 번 실행해보세요.
DLL 오류 / VC++ 관련 메시지가 떠요
  • Visual C++ 재배포 패키지 설치가 필요할 수 있습니다.
  • 운영자가 배포 링크를 제공하면 이 섹션에 추가하세요.
접속이 안 돼요 / 연결이 끊겨요
  • 서버 점검 중인지 서버점검 공지를 확인하세요.
  • 인터넷 환경/공유기 방화벽 설정을 확인하세요.
