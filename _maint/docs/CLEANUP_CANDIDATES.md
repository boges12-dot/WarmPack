# 정리 후보 리스트 (삭제는 선택)

이 문서는 **현재 코드/페이지에서 참조되지 않는 파일**을 찾아 “정리 후보”로 모아둔 목록입니다.

- 원칙: **바로 삭제하지 않고 후보로만 표시**
- 권장: 삭제 전, GitHub Pages에 올린 뒤 주요 동선(홈/다운로드/공지/커뮤니티) 1회 확인


## ✅ 정리 완료 (삭제 반영됨)
- `js/main.js` (미참조)
- `assets/icons/favicon-48x48.png` (미참조)

## 참고
- 이번 단계에서는 **미참조 파일만** 삭제했습니다.
- 만약 문제가 생기면, 이전 ZIP(auditfix8)로 언제든 롤백 가능합니다.

## 삭제 후 체크(권장)
1. 로컬에서 `index.html` 열기 → 메뉴/버튼 클릭으로 주요 페이지 이동
2. 공지/다운로드/커뮤니티 동선 1회 확인
3. `node _maint/tools/validate-data.mjs` 실행
