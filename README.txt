HYBRID SITE v2_3 (FULL)
======================

이 폴더는 '완성된 구조 뼈대'입니다.
- index.html 하나의 랜딩
- pages/ 아래에 메뉴별 폴더
- css/main.css, js/main.js 공통 사용
- 상단 메뉴/푸터는 자동으로 삽입됨(페이지마다 복붙 필요 없음)

사용 방법(초간단)
1) 이 폴더를 그대로 웹서버에 올리면 됩니다.
2) 로컬에서 확인하려면:
   - VS Code 설치 → 폴더 열기 → Live Server 확장 실행(권장)
   - 또는 아무 웹호스팅에 업로드

수정 방법(코드 몰라도 되는 범위)
- '페이지 내용'은 각 html 파일의 카드 영역 텍스트만 바꾸면 됨.
- 메뉴/위치 뼈대는 js/main.js가 담당(건드리지 말 것)

구조 핵심
- '서버 규정' 위치 고정: 소식 > 공지사항 하위
- 커뮤니티 규칙은 커뮤니티 내부(서버 규정과 분리)


[문제가 '다 이상하게' 보이는 경우]
- 보통 CSS/JS 경로가 안 잡힌 상태입니다.
- 이 RelPaths 버전은 각 페이지가 CSS/JS를 '상대경로'로 로드해서 더블클릭(file://)로 열어도 정상 스타일/드롭다운이 동작합니다.
- 단, 브라우저가 로컬 스크립트를 차단하면 Live Server(로컬 서버)로 여는 것이 가장 확실합니다.
